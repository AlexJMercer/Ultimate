O objetivo do blklist é criar uma TODO list/repositório de ideias/gerenciador de ideias para os membros blklight, essa ideia surge com a proposta de implementar o foi vista nas aulas ministrada por JoJo.
Desenvolver uma API em NodeJS usando AdonisJS para o back-end, para o front-end uso do Vue junto de bibliotecas de estilização e usando o axios para gerenciar as informações recebidas. Neste momento esta idéia se encontra em um estágio inicial de desenvolvimento.
